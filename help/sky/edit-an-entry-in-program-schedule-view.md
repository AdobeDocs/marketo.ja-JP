# プログラムスケジュール表示でのエントリの編集

<br> 

1. 編集する参加者を重複クリックします。 これは、どのカレンダー表示からでも実行できます。

   ![イメージ1](/help/sky/assets/program-schedule-view/edit-an-entry-in-program-schedule-view/edit-an-entry-in-program-schedule-view-1.png)

1. ポップアップで、エントリの様々な要素を編集できます。 スマートキャンペーンまたは電子メールプログラムのエントリの場合は、エントリのタイトルをクリックして、キャンペーン/プログラムの概要を開きます。

   ![イメージ2](/help/sky/assets/program-schedule-view/edit-an-entry-in-program-schedule-view/edit-an-entry-in-program-schedule-view-2.png)

>[!NOTE]
>
>基本エントリの名前は、一旦作成されると変更できません。

>[!NOTE]
>
>既に実行されているスマートキャンペーンと電子メールプログラムのエントリは、過去のものには移動できません。
