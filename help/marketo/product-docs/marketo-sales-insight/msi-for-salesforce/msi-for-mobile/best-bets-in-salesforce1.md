# [!DNL Salesforce1] における [!DNL Best Bets] {#best-bets-in-salesforce}

[[!DNL Best Bets]](/help/marketo/product-docs/marketo-sales-insight/msi-for-salesforce/features/stars-and-flames/priority-urgency-relative-score-and-best-bets.md) は、緊急度と相対的スコアが最も高いリードおよび連絡先です。 このリストに表示されるのは、自分が所有者であるリードだけで、リードのスコアが変化するとリストも更新されます。

1. [!DNL Salesforce] アプリのMarketo エリアに移動します。

   ドロップダウンで、「[!UICONTROL  ベストベット ]」、「[!UICONTROL  マイウォッチリスト ]」、「[!UICONTROL  ウェブアクティビティ ]」、「[!UICONTROL  匿名ウェブアクティビティ ]」または「[!UICONTROL  マイメール ]」を選択できます。
