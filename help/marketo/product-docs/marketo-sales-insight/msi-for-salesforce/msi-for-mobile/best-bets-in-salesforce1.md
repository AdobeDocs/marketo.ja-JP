# [!DNL Salesforce1] における [!DNL Best Bets] {#best-bets-in-salesforce}

[[!DNL Best Bets]](/help/marketo/product-docs/marketo-sales-insight/msi-for-salesforce/features/stars-and-flames/priority-urgency-relative-score-and-best-bets.md) は、緊急度スコアと相対スコアが最も高いリードと取引先責任者です。 このリストに表示されるのは、自分が所有者であるリードだけで、リードのスコアが変化するとリストも更新されます。

1. [!DNL Salesforce] アプリで、Marketo 領域に移動します。

   ドロップダウンで、「[!UICONTROL マイ最有望見込客]」、「[!UICONTROL マイウォッチリスト]」、「[!UICONTROL Web アクティビティ]」、「[!UICONTROL 匿名 web アクティビティ]」または「[!UICONTROL マイメール]」を選択できます。
