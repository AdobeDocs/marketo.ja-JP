# インサイトダッシュボード機能の概要 {#insights-dashboard-feature-overview}

セールスインサイトダッシュボードで使用できる機能の詳細をご覧ください。

>[!PREREQUISITES]
>
>最新の MSI SFDC パッケージと [設定](/help/marketo/product-docs/marketo-sales-insight/msi-for-salesforce/configuration/configuration-for-existing-customers.md).

![](assets/insights-dashboard-feature-overview-1.png)

## 取引先責任者のレイアウト {#contact-layout}

**エンゲージメント速度グリッド**

* このスマートグリッドには、過去 90 日間の注目のアクション、メール、ウェブアクティビティが含まれています
* ユーザーは、「アカウントを表示」アクティビティを選択できます。これにより、連絡先ビューで、すべてのアカウントレベルの注目のアクション、メール、ウェブアクティビティがレイヤー化されます
* 特定の週をハイライト表示して、その週内のアクティビティを確認できます
* デフォルトの表示：現在の週が選択されています

**エンゲージメントのドリルダウンと概要**

* 注目のアクティビティ、メール、ウェブアクティビティのアクティビティカードを掘り下げます
* 注目のアクティビティカード — 購読オプションが含まれます
* メールアクティビティカード — プレビューオプションを含む
* Web アクティビティカード — リンクをクリックする機能を含む
* その週の注目のアクション、メール、ウェブアクティビティが週別サマリバーに表示されます。 各アイコンはクリック可能で、特定のアクティビティを表示するフィルターとして使用できます
* デフォルトの表示：これは、現在のビューのアクティビティのリストです

**今後の電子メールキャンペーンおよびイベント**

「電子メールキャンペーン」タブ

* 今後 90 日以内にスケジュールされたメールプログラムまたはデフォルトプログラムの一部であるキャンペーンを含みます
* 連絡先/リードに固有のものではありません ( つまり、キャンペーンのリストは、Marketoインスタンスでスケジュールされたすべての電子メールキャンペーンの汎用リストです )。 すべてのリード、連絡先、アカウント、商談パネルにわたるキャンペーンのリストは同じになります
* キャンペーンの繰り返しが 90 日間に 3 回以上実行される場合、任意の時点で次の 3 回の実行のみが表示されます (Marketoの動作と同様 )。
* このセクションのアクティビティの詳細カードには、プレビューオプションが表示されます。 フローに複数の電子メール送信手順が含まれる場合、すべての電子メールをプレビューできます。 メール送信フロー手順内で、「メールの選択肢」が複数ある場合は、デフォルトのオプションをプレビューできます
* フィルターには、「検索」と「日付範囲」が含まれます

![](assets/insights-dashboard-feature-overview-2.png)

「イベント」タブ

* 今後 90 日以内にスケジュールされたイベントプログラムが含まれます
* すべてのイベント/招待イベントを（管理者設定に基づいて）表示するには、フィルターオプションを使用します。
* 招待イベントを選択すると、特定の連絡先が招待されたイベントとメンバーステータスが表示されます
* すべてのイベントを選択すると、90 日以内にスケジュールされたイベントのリストが表示されます
* このセクションのアクティビティの詳細カードには、プレビューオプションがあります
* フィルターには、「検索」、「招待されたイベントのみを表示」および「日付範囲」が含まれます

![](assets/insights-dashboard-feature-overview-3.png)

>
>アカウントまたは商談に 800 名を超える連絡先がある場合、ダッシュボードにはデータが表示されません。 ただし、個々の連絡先に移動して、そのインサイトとエンゲージメントを確認することができます。 お使いのアカウントに 800 名を超える連絡先がある場合、[ アカウントレベルのアクティビティの表示 ] は無効になります。
