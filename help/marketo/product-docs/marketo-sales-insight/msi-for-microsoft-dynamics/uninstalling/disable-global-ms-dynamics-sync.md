# グローバルMS Dynamics同期を無効にする{#disable-global-ms-dynamics-sync}

MS Dynamicsの同期を無効にするには、次の簡単な手順に従います。

1. Marketoで、**管理者**&#x200B;をクリックします。

   ![](assets/one.png)

1. [統合]で[**Microsoft Dynamics**]をクリックします。

   ![](assets/two.png)

1. 「**同期を無効にする**」をクリックします。

   ![](assets/three.png)

   >[!NOTE]
   >
   >インスタンスに「同期を無効にする」ボタンが表示されない場合は、[Marketoサポート](https://nation.marketo.com/t5/Support/ct-p/Support)にお問い合わせください。
