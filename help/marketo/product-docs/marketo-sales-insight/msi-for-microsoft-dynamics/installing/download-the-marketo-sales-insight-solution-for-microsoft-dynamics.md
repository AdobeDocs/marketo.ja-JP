# [!DNL Marketo Sales Insight] の [!DNL Microsoft Dynamics] ソリューションのダウンロード {#download-the-marketo-sales-insight-solution-for-microsoft-dynamics}

>[!NOTE]
>
>**管理者権限が必要**

>[!IMPORTANT]
>
>このページのプラグインは、Marketoのネイティブ CRM 同期ソリューションを使用してMarketo Engageと [!DNL Dynamics 365] に同期するユーザー向けです。 カスタム同期 [!DNL MS Dynamics 365 Online] （9.x 以降）を持ち、[!DNL Marketo Sales Insight] を購入しているユーザーの場合、[ パッケージはこちら ](https://mktg-cdn.marketo.com/community/MarketoSalesInsight_NonNative.zip){target="_blank"} です。

1. 「**[!UICONTROL 管理者]**」領域に移動します。

   ![](assets/mainnavhand.png)

1. 「**CRM**」をクリックします。

   ![](assets/image2015-3-11-13-3a7-3a11.png)

1. 「**Microsoft**」を選択します。

   ![](assets/image2016-5-3.png)

1. 「**[!UICONTROL Marketo ソリューションをダウンロード]**」を選択します。

   ![](assets/image2015-3-11-13-3a10-3a4.png)

1. [!DNL Microsoft Dynamics] バージョンに適したソリューションを選択します。

   ![](assets/msd-online.png)

これで完了です。ソリューションの zip ファイルがデバイスにダウンロードされます。
