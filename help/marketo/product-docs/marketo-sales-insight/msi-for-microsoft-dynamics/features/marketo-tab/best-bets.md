# 最優先 {#best-bets}

「ベストベット」タブには、優先度に基づいてすべてのホットリードのリストが表示され、緊急度と相対スコアを使用して計算されます。

「アクション」列の下の点メニューをクリックすると、次のようなエンゲージメントオプションを使用できます。
* Marketo メールの送信 
* Marketo キャンペーンに追加

「おすすめコンテンツ」タブから複数のリードを選択し、 _Marketo E メールの送信_ または _Marketo Campaign に追加_.
