# 最有望見込客 {#best-bets}

「最優先」タブには、優先度に基づいて計算されるすべてのホットリードのリストが表示されます。緊急度と相対スコアが使用されます。

「アクション」列の下のドットメニューをクリックすると、次のようなエンゲージメントオプションを使用できます。
* Marketo メールの送信
* Marketo キャンペーンに追加

また、[ 最優先 ] タブから複数のリードを選択し、 _Marketo E メールを送信_ または _Marketo Campaign に追加_.
