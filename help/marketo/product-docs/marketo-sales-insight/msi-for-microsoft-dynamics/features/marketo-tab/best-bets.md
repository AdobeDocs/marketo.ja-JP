# [!DNL Best Bets] {#best-bets}

「[!UICONTROL 最有望見込客]」タブには、緊急度と相対スコアを使用して計算された優先度に基づいて、有望なリードのリストが表示されます。

「アクション」列の下のドットメニューをクリックすると、次のようなエンゲージメントオプションを使用できます。

* [!UICONTROL Marketo メールを送信]
* [!UICONTROL Marketo キャンペーンに追加]

また、「[!DNL Best Bets]」タブから複数のリードを選択し、「_[!UICONTROL Marketo メールを送信]_」または「_[!UICONTROL Marketo キャンペーンに追加]_」を選択できます。
