# Sales Insight アクションユーザーオンボーディングチェックリスト {#sales-insight-actions-user-onboarding-checklist}

オンボーディングチェックリストは、新しい Sales Insight Actions インスタンスの有効化、構成、オンボーディングを調整する際に役立ちます。 設定を完了するには、Marketo、Salesforce、Marketo Sales Insight のアクションを更新する必要があります。そのため、各システムの管理者と連携する必要があります。

以下の画像をクリックして、チェックリストを表示およびダウンロードします。
