# サーバーで [!DNL Sales Insight Actions] を許可リストに加える方法を教えてください。 {#how-can-i-allowlist-sales-insight-actions-on-my-server}

残念ながら、1 つの静的 IP アドレスを提供する方法はありません。[!DNL Sales Insight Actions] はクラウドにデプロイされ、200 を超えるサーバーを利用してメールやユーザリクエストを処理します。非常に広い範囲が提供されるので、これ以上の安心感はありません。
