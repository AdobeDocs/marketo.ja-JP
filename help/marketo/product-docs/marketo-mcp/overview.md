# MCPの概要ドキュメント {#overview}

ATHERのテキスト：

Marketo Engageのパブリック MCP Serverは、Marketo REST APIをAI エージェントに構造化ツールとしてアクセスできるようにする、クラウドベースのサービスです。 MCP （Model Context Protocol）上に構築されており、標準化されたスキーマ駆動型のインターフェイスを提供し、AI システムがMarketoを安全かつ効率的に利用できるようにします。

MCP Serverは、Marketoのエンドポイントを各アプリケーションに手動で統合する代わりに、AI ネイティブな単一のゲートウェイを提供し、MCP互換のエージェントが利用可能なオペレーションを発見し、必要なパラメーターを理解し、一貫したプロトコルを通じてアクションを実行できるようにします。

パブリック REST APIと内部APIの両方で構成されるツールのホストがあり、UIと同様にMarketo コア関数を実行しますが、API呼び出しを通じて実行します。 ツールが開発されれば、さらに多くのツールが継続的に追加されます。 &lt;明確化 – -- – 必要性

MCP Serverはセルフサービスとして提供され、エンドポイントを構築して接続を認証することで、Adobe IO フレームワークを通じて設定できます。 詳細と設定の詳細については、Adobe Developerのサイトをご覧ください（リンクを追加するには、こちらを参照）。 件名の？ https://developer.adobe.com/marketo-apis/

SOAP APIとして表示されるCampaign ツールは、REST APIに変換され、追加されました。

設定が完了したら、使用可能なツールのリストを表示するには、こちらの手順に従います（ハイパーリンク）。

メモ：使用できないツールは、「Campaignの非アクティブ化」と「静的リストの削除」（未定）です。
