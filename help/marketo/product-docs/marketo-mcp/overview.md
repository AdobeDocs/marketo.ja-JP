# MCPの概要ドキュメント {#overview}

ATHERのテキスト：

Public MCP Server for Marketo Engageは、Marketo Engageのパブリック REST APIの全機能を、AI エージェント用の構造化された検出可能なツールとして公開する、クラウドアクセス可能なサービスです。 Model Context Protocol （MCP）上に構築され、Marketoのマーケティングオートメーション能力を、AI システムが安全かつインテリジェントに使用できる、標準化されたスキーマドリブン型のインターフェイスに変換します。

MCP Serverは、Marketoのエンドポイントを各アプリケーションに手動で統合する代わりに、単一のAI ネイティブゲートウェイとして機能し、MCP互換エージェントが使用可能なオペレーションを発見し、必要なパラメーターを理解し、一貫したプロトコルを通じてアクションを実行できるようにします。

パブリック REST APIと内部APIの両方で構成されるツールのホストがあり、UIと同様にMarketo コア関数を実行しますが、API呼び出しを通じて実行します。 ツールが開発されれば、さらに多くのツールが継続的に追加されます。

MCP Serverはセルフサービスとして提供され、エンドポイントを構築して接続を認証することで、Adobe IO フレームワークを介してセットアップできます。 詳細および設定の詳細については、Adobe Developer サイト（リンクを追加）を参照してください。

SOAP APIとして表示されるCampaign ツールは、REST APIに変換され、追加されました。

設定が完了したら、使用可能なツールのリストを表示するには、こちらの手順に従います（ハイパーリンク）。

メモ：使用できないツールは、「Campaignの非アクティブ化」と「静的リストの削除」（未定）です。
