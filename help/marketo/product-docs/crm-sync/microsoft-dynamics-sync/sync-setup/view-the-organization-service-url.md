# 組織サービス URL の表示 {#view-the-organization-service-url}

Marketo Engageを MD インスタンスと同期するには、組織サービス URL が必要です。 Dynamics で検索する方法を次に示します。

1. [!DNL Dynamics] にログインします。「設定」アイコンをクリックし、「**[!UICONTROL 詳細設定]**」を選択します。

   ![](assets/one.png)

1. 「**[!UICONTROL 設定]**」を選択し、「**[!UICONTROL カスタマイズ]**」を選択します。

   ![](assets/two.png)

1. 「**[!UICONTROL 開発者リソース]**」をクリックします。

   ![](assets/three.png)

1. 組織サービス URL は、「**[!UICONTROL サービスエンドポイント]**」にあります。

   ![](assets/four.png)

1. その URL を Marketo にコピー＆ペーストして同期します。
