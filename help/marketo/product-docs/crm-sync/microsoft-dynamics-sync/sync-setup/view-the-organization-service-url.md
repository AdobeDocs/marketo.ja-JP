# 組織サービス URL の表示 {#view-the-organization-service-url}

Marketo を Dynamics インスタンスと同期するには、組織サービス URL が必要です。Dynamics で検索する方法を次に示します。

1. Dynamics にログインします。「設定」アイコンをクリックし、「**詳細設定**」を選択します。

   ![](assets/one.png)

1. 「**設定**」を選択し、「**カスタマイズ**」を選択します。

   ![](assets/two.png)

1. 「**開発者リソース**」をクリックします。

   ![](assets/three.png)

1. 組織サービス URL は、「**サービスエンドポイント**」にあります。

   ![](assets/four.png)

1. その URL を Marketo にコピー＆ペーストして同期します。
