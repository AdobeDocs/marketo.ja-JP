# ランディングページへの SSL の追加 {#add-ssl-to-your-landing-pages}

Marketo Engageで作成されたランディングページにブランドドメインのエイリアス（`http://business.adobe.com` など）を追加して、ブランドドメインの下でアクセスできるようにする方法を説明します。

スクリーンショット

## SSL 証明書を有効にする {#enable-ssl-certification}

ランディングページのルールの一部として作成したすべてのドメインエイリアスに SSL を自動的に追加します。

1. 「**管理者**」領域に移動します。

   スクリーンショット

1. ツリーから **ランディングページ** を選択します。 「**ルール**」タブで「**新規**」ドロップダウンをクリックし、「**新しいドメインエイリアス**」を選択します。

   スクリーンショット

1. 「**SSL 証明書を生成**」チェックボックスを選択します。

   スクリーンショット

このドメインの SSL 証明書が自動的に追加されます。

スクリーンショット

スクリーンショット

メモ：

リストの「SSL 証明書」列には、この機能のリリース後に作成されたすべてのドメインエイリアスの SSL 証明書ステータスが表示されます。 サポートを通じてドメインに対して SSL を有効にしている場合、それぞれの SSL 証明書は引き続き存在しますが、この表には、この機能を使用して追加されたドメインの SSL 証明書のみが反映されます

SSL が準備完了状態になるまで最大 3 分かかる場合があり、UI に変更が反映されるようにページを更新する必要があります。
