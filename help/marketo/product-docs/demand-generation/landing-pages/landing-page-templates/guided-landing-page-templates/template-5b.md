# テンプレート 5B {#template-5b}

![](assets/template-5b.png)

このテンプレートには、次の内容が含まれます。

* ヘッダー（オプション）
* プライマリセクション

   * ヒーロータイトルおよびヒーローテキストが含まれます。

* 5 つの本文セクション（オプション）
* フッター（オプション）
