# テンプレート 11A {#template-11a}

右クリックして[テンプレート 11A をダウンロード](https://experienceleague.adobe.com/landing/marketo/lp-templates/template-11a.html)します

![](assets/image2015-8-4-9-3a56-3a23.png)

このテンプレートには、次の内容が含まれます。

* ヘッダー（オプション）
* プライマリセクション

   * ヒーロータイトル、ヒーローテキストおよびフォームが含まれます

* 4 つの本文セクション（オプション）
* フッター（オプション）

**このテンプレートをダウンロードするには、以下を右クリックします。**

[Template 11A.html](https://experienceleague.adobe.com/landing/marketo/lp-templates/template-11a.html)
