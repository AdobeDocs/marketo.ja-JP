# テンプレート 7D {#template-7d}

右クリックして[テンプレート 7D をダウンロード](https://experienceleague.adobe.com/landing/marketo/lp-templates/template-7d.html?lang=ja)します

![](assets/image2015-7-29-15-3a5-3a39.png)

このテンプレートには、次の内容が含まれます。

* ヘッダー（オプション）
* プライマリセクション

   * ヘッダーおよびフォームが含まれます

* 4 つの本文セクション（オプション）
* フッター（オプション）
