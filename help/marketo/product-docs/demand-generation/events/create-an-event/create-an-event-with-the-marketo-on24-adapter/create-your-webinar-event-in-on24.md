# ON24でウェビナーイベントを作成する{#create-your-webinar-event-in-on}

ON24でウェビナーを作成した後、Marketoイベントを作成し、それをON24ウェビナーに関連付けることができます。 これにより、システムは登録情報と出欠情報を共有できます。 確認用の電子メールやICSファイルで使用するURLやその他の情報をメモしておきます。

ON24ウェビナーの作成時には、Marketoが使用する設定と、ON24が使用する設定があります。 Marketoは次の設定を使用します。

**タイトルと説明**

* イベントのタイトル — ウェビナーの名前。 これはMarketoで見ることができます。
* イベントの抜粋（オプション） — ウェビナーの説明。 これはMarketoで見ることができます。

**日時**

* ライブイベント開始-開始の日時。 これはMarketoで見ることができます。
* Live End Time — 終了日時。 これはMarketoで見ることができます。

次のステップは、[Marketo](/help/marketo/product-docs/demand-generation/events/create-an-event/create-an-event-with-the-marketo-on24-adapter/create-an-event-in-marketo.md)にイベントを作成することです。
