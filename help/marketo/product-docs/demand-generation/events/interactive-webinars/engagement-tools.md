# インタラクティブウェビナーのエンゲージメントツール {#engagement-tools-in-interactive-webinars}

Adobe Connectルーム内のポッド配信機能。 ポッドをレイアウトに追加するには、ポッドメニューをクリックし、追加するポッドを選択します。

スクリーンショット

## 利用可能なポッド {#available-pods}

### 共有 {#share}

共有ポッドは、部屋のコンテンツを共有するために使用されます。 画面、ドキュメント、ホワイトボードの共有に使用できます。 共有ポッドで共有および表示できるファイルタイプは、プレゼンテーション（PPT、PPTX）、ドキュメント（PDF）、オーディオ（MP3）、ビデオ（MP4）、画像（JPG、PNG）、カスタムポッド（POD、ZIP）です。 アップロードされたコンテンツは部屋内に保持され、すぐに利用できるようになります。 Share Podについて詳しくは、このAdobe ヘルプ記事とビデオをご覧ください。

スクリーンショット

メモ

メモ ポッドは、ホストとプレゼンターが情報を共有し、部屋でメモを取るために使用します。 リッチテキスト形式をサポートしており、メモはポッドのメニューから（ホスト別に）書き出すこともできます。 Notes ポッドについて詳しくは、このAdobe ヘルプ記事とビデオをご覧ください。

スクリーンショット

参加者

参加者ポッドは、イベント中の参加者、そのアクティビティ、および権限を管理するために使用されます。 主催者やプレゼンターは、参加者の名前、役割、ステータス（挙手や同意など）を監視できます。 ポッドは、ブレイクアウトセッションの管理にも使用されます。 参加者ポッドについて詳しくは、このAdobeのヘルプ記事とビデオをご覧ください。

スクリーンショット

動画

ビデオポッドは、web カメラのフィードを共有するために使用されます。 Web カメラは、グリッド、フィルムストリップ、または単一のスピーカービューに配置できます。 ビデオポッドについて詳しくは、このAdobeのヘルプ記事を参照してください。

スクリーンショット

チャット

チャットポッドは、ルーム内の他の参加者とのコミュニケーションに使用されます。 チャットポッド内のチャットは、削除されるまで保持されます。 チャットはホストに電子メールで送信して保存できます。 参加者は、チャットポッド内のチャットのフォントサイズを調整できます。

ホストとプレゼンター領域に追加されたチャットポッドは、プレゼンテーションチームと非公開でコミュニケーションを取ることができます。 Chat Podについて詳しくは、このAdobeのヘルプ記事とビデオをご覧ください。

注：チャットポッドは常に記録されます。 チャットを録音に含めたくない場合は、チャットパネルを使用します。 Chat Podについて詳しくは、Adobeのヘルプ記事をご覧ください。 [新しい配信インタラクティブウェビナーページ ]の「記録」セクションの「チャット」にリンクします。

スクリーンショット

Q&amp;A

Q&amp;A （質問と回答）ポッドは、イベント中にモデレートされた議論を行うために使用されます。 参加者は、自分が質問した質問と、ホストまたはプレゼンターによる質問に対する回答のみを表示できます。 ただし、質問が公開された場合、その質問と回答の両方が全員に表示されます。

ホストは、会議室の環境設定から送信者とプレゼンターの名前を非表示にすることができます。

スクリーンショット

Q&amp;A ポッドのプレゼンテータービューをホストとプレゼンテーターエリアに移動して、質問を調整するために大きな不動産を利用できるようにすることもできます。

スクリーンショット

Q&amp;A ポッドの詳細については、Adobeのヘルプ記事および動画をご覧ください。

スクリーンショット

投票

投票ポッドは、セッション中に参加者から質問を行うために使用されます。 投票ポッドは、複数選択、複数回答、および簡潔な回答タイプの質問をサポートしています。 主催者は、結果を参加者にブロードキャストできます。 Poll Podについて詳しくは、このAdobeのヘルプ記事とビデオをご覧ください。

スクリーンショット

ファイル

ファイルポッドは、ダウンロード可能なコンテンツを共有するために使用されます。 これは、製品ホワイトペーパー、ケーススタディ、プレゼンテーション、ビデオなどを参加者と共有するために使用できます。 ファイルは、プラスボタンをクリックするか、ファイルをポッドに直接ドラッグ&amp;ドロップすることで追加できます。 ファイルポッドについて詳しくは、このビデオをご覧ください。

スクリーンショット

Web リンク

Web リンクポッドは、Web URLを共有するために使用されます。 これは、ソーシャルハンドル、製品コミュニティページ、ホワイトペーパー、ケーススタディなどのリンクを参加者と共有するために使用できます。

主催者は、参加者にリンクをプッシュして、そのリンクをブラウザーウィンドウで開くこともできます。

Web リンクポッドについて詳しくは、このビデオをご覧ください。

スクリーンショット

エンゲージメントダッシュボード

エンゲージメントダッシュボードは、ホストとプレゼンターがセッション中の参加者の注意とインタラクティビティレベルを監視するために使用します。 エンゲージメントダッシュボードには、ホストとプレゼンテーターエリア内でのみアクセスできます。 エンゲージメントダッシュボードの詳細については、動画をご覧ください。

スクリーンショット

### カスタムポッド {#custom-pods}

カスタムポッドは、Adobe Connectが提供する機能を拡張します。 無料で利用できる人気のあるカスタムポッドには、カウントダウンタイマー、クロック、ハウスキーパー、タイトラー、Word Cloudなどがあります。 カスタムポッドは、<https://apps.adobeconnect.com/>から.podまたは.zip ファイルとしてダウンロードできます。

カスタムポッドを使用するには：

1. 新しい共有ポッドを追加します。
1. 「文書」をクリックします。
1. ダウンロードした.podまたは.zip ファイルをアップロードし、「共有」をクリックします。

## Beyond the Basics {#beyond-the-basics}

### 分類の管理 {#managing-breakouts}

ブレイクアウトルームは、ミーティングまたはトレーニングセッション内で作成できるサブルームです。 大規模なグループを、話し合いや共同作業が可能な小さなグループに分割するのに便利です。 ホストは、1つのセッションに対して最大20個のブレイクアウトルームを作成できます。

ブレイクアウトルームでは、参加者は話したり、web カメラを共有したり、チャットポッドを使用したり、ホワイトボードで共同作業したり、画面を共有したりすることができます。 ホストは、すべてのブレイクアウトルームにアクセスして、質問を支援し、回答することができます。

ブレイクアウトルームでの作業が完了すると、ホストはブレイクアウトセッションを終了し、参加者をメインルームに戻すことができます。 主催者は、ブレイクアウトルームで何が起こったかを全員と共有してレビューすることができます。 部屋を再度使用する場合は、ブレークアウト ルームのレイアウトとコンテンツを再利用することもできます。

ブレイクアウトルームについて詳しくは、このプレイリストのビデオをご覧ください。

### ポッドからコンテンツを消去 {#clear-content-from-pods}

POD メニューの「ポッドからコンテンツをクリア」オプションを使用すると、新しいセッションのために部屋を再利用する際に、ポッド内からコンテンツをクリアできます。 これにより、部屋の中からのみコンテンツがクリアされます。 過去のセッションのインタラクションデータは削除されず、分析のために引き続きMarketoで使用できます。

### ポッド名の変更 {#renaming-pods}

メモ、チャット、ファイル、Web リンク、共有などの一部のポッドでは、部屋に複数のインスタンスを持つことができ、複数のレイアウトでポッドのインスタンスを再利用できます。 デフォルトでは、特定のタイプのポッドの複数のインスタンスを作成する場合、名前に番号が追加され、異なるインスタンスを区別するのに役立ちます。 ただし、ホストはこのようなポッドの名前を簡単に変更できます。

ポッドの名前を変更するには、ポッドタイトル内の名前をダブルクリックし、新しい名前を入力します。 これを説明しているこのビデオをご覧ください。

### ポッドの切り替え {#switching-pods}

ホストが同じレイアウトを維持しながらポッドを簡単に切り替える方法があります。 レイアウトを切り替えるには、ポッドタイトルのドロップダウンアイコンをクリックして選択します。 変更は即座に参加者に反映されます。 このオプションは、複数のインスタンス（チャット、メモ、投票、共有、ファイル共有、Weblinks、クイズ）をサポートするすべてのポッドで使用できます。

スクリーンショット

レイアウトの作成と管理

レイアウトパネルは、部屋内のレイアウトを管理するために使用されます。 部屋の中のホストにしか見えません。 パネルを使用して、レイアウトの作成、削除、複製、名前変更、並べ替えをおこなうことができます。 レイアウトの管理について詳しくは、この動画をご覧ください。

ルーム設定と環境設定

部屋の環境設定にアクセスするには、部屋メニューから「環境設定」オプションを選択します。 環境設定ダイアログ内の設定は、ホストがセッション内のエクスペリエンスを制御するのに役立ちます。 さまざまな環境設定と、それらが部屋の動作にどのように影響するかを知るには、このビデオを参照してください。
