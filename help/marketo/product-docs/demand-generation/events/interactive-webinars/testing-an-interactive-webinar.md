# インタラクティブウェビナーのテスト {#testing-an-interactive-webinar}

インタラクティブウェビナーでは、Adobe Connectをウェビナープロバイダーツールとして使用します。 最初のライブウェビナーを開催する前に、テストウェビナーを作成することをお勧めします。

## ワークフロー {#workflow}

インタラクティブウェビナーは、テストウェビナーとして使用できます。 テストは、10人以下のホストとプレゼンター（参加者）を追加することによって実行できます。

テストウェビナーを作成するワークフローは、インタラクティブウェビナーイベントプログラムに関して、他のウェビナー[&#128279;](/help/marketo/product-docs/demand-generation/events/interactive-webinars/create-an-interactive-webinar.md)を作成すると同じままです。 配達に選択した日付は将来の日付に設定することをお勧めします。

>[!CAUTION]
>
>テストウェビナーの日時が近すぎると、その日時が経過した場合、インタラクティブウェビナーは配信されたウェビナーを考慮し、ライセンスから1 クレジットを差し引きます。

テストウェビナーでは1つのクレジットがブロックされますが、実際のウェビナーで使用するようにプログラムをスケジュール変更するか、テストウェビナープログラムを削除することで使用できます。

ウェビナーを作成およびデザインした後、ホスト、共同ホスト、および/またはプレゼンターが参加して配信をリハーサルできます。 外部の招待者を共同ホストとして追加して、テストに参加できるようにすることもできます。 テストは、ウェビナーが配信されるのと同じ方法で実行できるため、評価者はウェビナーが実際のオーディエンスにどのように配信されるのかを理解できます。 ライブウェビナーのすべての機能は、テストウェビナーでも使用できます。 これは、ウェビナーの開始時間の15分前まで実行できます。

## テスト時に使用できない機能 {#features-not-available-when-testing}

これはテストウェビナーであるため、通常ライブウェビナーの後に行われるもので、利用できないものがいくつかあります。 これには、ウェビナーの録画、エンゲージメントダッシュボードの形式でのウェビナーのエンゲージメントインサイトの取得、またはダッシュボードタブの集約ダッシュボードが含まれます。 録画されたウェビナーで使用できるインタラクティブウェビナーのオンデマンドウェビナーのトラッキングも使用できません。 テストウェビナー中はエンゲージメントデータが追跡されないため、Adobe ConnectからMarketo Engageにデータが返されないため、ウェビナー後のナーチャリングは実行できません。
