# インタラクティブウェビナーの基本を学ぶ {#get-started-with-interactive-webinars}

そこで、[&#x200B; インタラクティブウェビナーの概要](/help/marketo/product-docs/demand-generation/events/interactive-webinars/interactive-webinars-overview.md){target="_blank"}{target="_blank"}を読み、次に進む準備ができました。 次の手順に従って開始してください。

![](assets/get-started-with-interactive-webinars-1.png)

## 今後の施策 {#what-to-do}

**手順1**: [&#x200B; ユーザーを追加](/help/marketo/product-docs/demand-generation/events/interactive-webinars/user-and-license-management.md#add-a-user){target="_blank"} — インタラクティブ ウェビナーへのアクセス用にユーザーを追加（後で削除）する方法を説明します。

**手順2**: [&#x200B; インタラクティブウェビナーを作成](/help/marketo/product-docs/demand-generation/events/interactive-webinars/create-an-interactive-webinar.md){target="_blank"}{target="_blank"} — イベントプログラムの作成方法と、ウェビナーの初期パラメーターの設定方法について説明します。

**ステップ 3 （オプション）**: [&#x200B; ウェビナーチームを追加](/help/marketo/product-docs/demand-generation/events/interactive-webinars/add-a-webinar-team.md){target="_blank"}{target="_blank"} – 自分でウェビナーを開催する予定がない場合は、プレゼンターと共同ホストを追加してホスティング/プレゼンターの職務を共有する方法を説明します。

**ステップ 4**: [&#x200B; インタラクティブウェビナーのデザイン &#x200B;](/help/marketo/product-docs/demand-generation/events/interactive-webinars/designing-interactive-webinars.md){target="_blank"} — ウェビナーが作成され、オプションのチームが追加されたら、特定のニーズに合わせてイベントを調整する必要があります。

**ステップ 5**: [&#x200B; ウェビナーを宣伝する](/help/marketo/product-docs/demand-generation/events/interactive-webinars/promoting-an-interactive-webinar.md){target="_blank"} – 今後のイベントを宣伝する方法に関する貴重なヒントを学びます。

**ステップ 6**: [&#x200B; インタラクティブウェビナーを配信](/help/marketo/product-docs/demand-generation/events/interactive-webinars/deliver-an-interactive-webinar.md){target="_blank"} — ウェビナーのプレゼンテーションについて知っておくべきあらゆることを説明します。

## その他のドキュメント {#additional-documentation}

ウェビナーが終了したら、イベントの概要画面で便利なダッシュボードを使用して、そのパフォーマンスを確認します。 オーディエンスのエンゲージメントの詳細は、[&#x200B; エンゲージメントダッシュボード &#x200B;](/help/marketo/product-docs/demand-generation/events/interactive-webinars/engagement-dashboard.md){target="_blank"}で確認できます。 ウェビナーのエンゲージメントを活用して、[&#x200B; イベントワークフロー](/help/marketo/product-docs/demand-generation/events/interactive-webinars/event-workflows.md){target="_blank"}のスマートフィルターとトリガーを使用して、パーソナライズされた方法でオーディエンスをターゲティングします。 また、[&#x200B; オンデマンドウェビナー](/help/marketo/product-docs/demand-generation/events/interactive-webinars/on-demand-webinars.md){target="_blank"}を通じてオンデマンドパフォーマンスを追跡する方法について説明します。

このトピックに関するヘルプ記事には、[&#x200B; インタラクティブウェビナーのベストプラクティス &#x200B;](/help/marketo/product-docs/demand-generation/events/interactive-webinars/best-practices-for-interactive-webinars.md){target="_blank"}、[あなたのニーズ/ブランディングを反映させるためにウェビナー](/help/marketo/product-docs/demand-generation/events/interactive-webinars/customization.md){target="_blank"}をカスタマイズする方法、[&#x200B; インタラクティブウェビナーのテンプレート &#x200B;](/help/marketo/product-docs/demand-generation/events/interactive-webinars/templates.md){target="_blank"}を使用して組織全体のテンプレートを作成する方法、[&#x200B; ルーム管理](/help/marketo/product-docs/demand-generation/events/interactive-webinars/room-management.md){target="_blank"}でルームと録画を管理する方法などがあります。

 

インタラクティブウェビナー（Adobe Connectを活用）をお楽しみください。
