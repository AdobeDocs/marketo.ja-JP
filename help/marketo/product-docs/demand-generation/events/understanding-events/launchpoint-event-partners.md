# LaunchPointイベントパートナー{#launchpoint-event-partners}

MarketoをLaunchPointイベントパートナーに接続して、オンラインイベントを自動化します。 接続すると、次のことができます。

* ウェビナーイベントの管理
* Marketoランディングページでキャプチャされた登録情報を、設定済みのLaunchPointイベントパートナーにプッシュします。
* 出席情報を自動的にMarketoイベントに取り込む
* プログラムのメンバーとしての人のステータスの取得

オンラインのMarketoイベントとLaunchPointイベントパートナーを統合する手順を参照してください。

* [Adobe接続](/help/marketo/product-docs/demand-generation/events/create-an-event/create-an-event-with-adobe-connect.md)
* [ON24](/help/marketo/product-docs/demand-generation/events/create-an-event/create-an-event-with-the-marketo-on24-adapter/create-your-webinar-event-in-on24.md)
* [ReadyTalk](/help/marketo/product-docs/demand-generation/events/create-an-event/create-an-event-with-readytalk.md)
