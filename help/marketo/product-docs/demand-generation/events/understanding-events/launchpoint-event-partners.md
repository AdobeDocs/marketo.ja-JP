# LaunchPointイベントパートナー{#launchpoint-event-partners}

MarketorをLaunchPointイベントパートナーに接続して、オンラインイベントを自動化します。 接続すると、次のことができます。

* ウェビナーイベントの管理
* Marketorランディングページでキャプチャされた登録情報を、設定済みのLaunchPointイベントパートナーにプッシュします。
* 出席情報を自動的にマーケティング担当者のイベントに取り込む
* プログラムのメンバーとしての人のステータスの取得

オンラインマーケティングイベントとLaunchPointイベントパートナーを統合する手順を参照してください。
