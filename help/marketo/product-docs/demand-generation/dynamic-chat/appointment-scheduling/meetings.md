# 会議 {#meetings}

Web サイトの訪問者が様々なダイアログを使用してスケジュールしたすべての予定が表示されます。ここには、予約者のメールアドレス、予約したエージェント、予約の日時、予定した会議時間が過ぎたかどうかが表示されます。
