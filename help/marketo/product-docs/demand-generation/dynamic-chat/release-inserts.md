## Marketo Engageスマートリストターゲティング {#marketo-engage-smart-list-targeting}

マーケターは、ダイアログのオーディエンス条件でMarketo Engageスマートリストのメンバーをターゲット設定できます。

## 新しいDynamic Chatアクティビティ {#new-dynamic-chat-activities}

対話フロー、ライブチャット、その他の新機能をサポートする新しいアクティビティ。

## 新しいアクティビティのトリガートークン {#trigger-tokens-for-new-activities}

新しいトリガートークンは、Dynamic Chatの新しいアクティビティをサポートするために使用できます。


JOHN

## ライブチャット {#live-chat}

Web 訪問者は、リアルタイムでセールス担当者とチャットして、質問に回答を得ることができます。

## チーム {#teams}

ライブチャットや会議の予約用に訪問者をルーティングできるエージェントのチームを作成します。

## リアルタイムライブチャット通知 {#real-time-live-chat-notifications}

訪問者がライブチャットをリクエストすると、エージェントはリアルタイムのアプリ内通知とブラウザー通知を受け取ります。

## フォールバック設定 {#fallback-settings}

管理者は、カスタムフォールバック設定を設定して、エージェントがチャットに使用できない場合に訪問者に表示する内容を定義できます。

## ユーザー設定の会議予約設定 {#custom-meeting-booking-settings}

エージェントは、使用可否、会議の長さ、会議の間の「バッファ時間」など、カスタムの会議予約設定を設定できます。


スレカンス

## 会話 SDK {#conversations-sdk}

任意のフォーム、ランディングページ、またはほとんどすべての Web ページを、魅力的な会議予約エクスペリエンスに変換します。

## グローバルパフォーマンスダッシュボード {#global-performance-dashboard}

エンゲージメントとパフォーマンスの指標（合計および長期）、パフォーマンスの高いページ、パフォーマンスの高いダイアログなど、ダイアログのパフォーマンスがどの程度良いかを確認できます。

## 会議パフォーマンスダッシュボード {#meeting-performance-dashboard}

予約されているミーティングの数と、予約されているミーティングの人数を確認します。

## Live Chat パフォーマンスダッシュボード {#live-chat-performance-dashboard}

ライブセールスエージェントが持つ会話の数、および最もパフォーマンスの高いチームを表示します。

## ストリームデザイナーのアクションカード {#action-card-in-stream-designer}

マーケターは、ダイアログストリームでアクションカードを使用して、訪問者がダイアログと関わる際に非表示の値をMarketo Engage人物レコードに渡すことができます。
