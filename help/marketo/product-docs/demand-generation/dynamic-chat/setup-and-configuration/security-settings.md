# セキュリティ設定 {#security-settings}

セキュリティ設定では、ブロックされたドメインまたは許可リストにドメインを追加できます。

![](assets/security-settings-1.png)

## ブロックされた電子メールドメイン {#blocked-email-domains}

エージェントにやり取りしてもらいたくないメールドメインを持つ訪問者（競合他社など）がいる場合は、そのメールドメインをブロックリストに追加します。

1. **検証を有効にする** スライダーを選択して、ブロックリストをアクティブにします。 最大 50 個のドメインを入力し、「**保存**」をクリックします。

   ![](assets/security-settings-2.png)

## 許可されたドメイン {#allowed-domains}

許可されたドメインを追加すると、サードパーティがサイトから JavaScript を削除して、自社のサイトに追加できなくなります。

1. **検証を有効にする** スライダーを選択して、許可リストをアクティブにします。 許可されたドメインを入力し、「**保存**」をクリックします。

   ![](assets/security-settings-3.png)
