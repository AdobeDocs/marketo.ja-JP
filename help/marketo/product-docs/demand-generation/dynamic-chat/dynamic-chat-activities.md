# 動的チャットのアクティビティ {#dynamic-chat-activities}

動的チャットスマートリストで使用する複数のフィルターとトリガーを提供します。

![](assets/dynamic-chat-activities-1.png)

## 注意事項 {#things-to-note}

* 条件は、動的チャットのフローステップでサポートされます
* 動的チャットアクティビティは、[Marketo Sales Insight](/help/marketo/product-docs/marketo-sales-insight/msi-for-salesforce/features/dynamic-chat-integration.md){target="_blank"} と同期されます
* 人物レコードのアクティビティログで、個々の動的チャットアクティビティを表示できます
