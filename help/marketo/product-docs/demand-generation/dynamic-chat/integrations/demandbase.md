# Demandbase {#demandbase}

Demandbase ユーザーは、Demandbase の人物属性を使用して、Dynamic Chatでのダイアログターゲティング、条件付きブランディングおよびカスタムルーティングを行うことができます。

**統合機能**

ダイアログや会話のフローを作成しながら、ネイティブ属性とカスタム属性に加えて、Demandbase 属性に基づいてオーディエンスをターゲットに設定します。

![](assets/demandbase-1.png)

Demandbase 属性を、条件付きブランチ、ダイアログまたは会話フローの条件として使用します。

![](assets/demandbase-2.png)

カスタムルーティングロジックを定義する際は、Demandbase Attributes を使用します。

![](assets/demandbase-3.png)
