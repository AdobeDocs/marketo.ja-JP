# Adobe Experience Cloud統合の概要{#adobe-experience-cloud-integrations-overview}

Adobeは、Adobe Experience Cloud製品全体でネイティブオーディエンスを同期することから始まり、堅牢なデマンドマーケティングソリューションを提供します。 Adobe Audience Manager全体で独自の B2B 中心のデータを活用し、強力な需要マーケティングの使用例と、既存のAdobe・アプリケーションに対する未活用の潜在力を解放します。

>[!NOTE]
>
>統合 (Audience ManagerとAdobe Real-Time CDP Marketo Connector を含む ) では、Marketo Engage内の IMS 組織のマッピングが必要になる場合があります。 [組織のマッピングを設定するAdobeの詳細](/help/marketo/product-docs/adobe-experience-cloud-integrations/set-up-adobe-organization-mapping.md){target=&quot;_blank&quot;}。

**MarketoとAdobe Audience Managerの統合**

* [Adobe Experience Cloudからのオーディエンスの同期](/help/marketo/product-docs/adobe-experience-cloud-integrations/sync-an-audience-from-adobe-experience-cloud.md){target=&quot;_blank&quot;}

* [リストをAdobe Experience Cloudに送信](/help/marketo/product-docs/core-marketo-concepts/smart-lists-and-static-lists/static-lists/send-a-list-to-adobe-experience-cloud.md){target=&quot;_blank&quot;}

Marketo Engageから複数の AEC アプリケーションに既知の担当者の静的リストを同期できます。これにより、既存のプログラムの強化、新しい使用例のロック解除、マルチチャネルキャンペーンの調整が可能になります。 この統合には、以下が含まれます。Adobe Analytics、Adobe Target、Adobe Experience Manager、Adobe Audience Manager、Adobe Advertising Cloud
