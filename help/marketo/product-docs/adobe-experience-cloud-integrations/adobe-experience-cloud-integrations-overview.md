# Adobe Experience Cloud統合の概要{#adobe-experience-cloud-integrations-overview}

Adobeは、Adobe Experience Cloud製品全体でネイティブオーディエンスを同期することから始まり、堅牢なデマンドマーケティングソリューションを提供します。 Marketo Engageは、一部の製品と統合されています。 Adobe Audience Manager全体で独自の B2B 中心のデータを活用し、強力な需要マーケティングの使用例と、既存のAdobe・アプリケーションに対する未活用の潜在力を解放します。

>[!NOTE]
>
>Audience Manager、B2B CDP Marketoコネクタ、Dynamic Chat を含む統合では、Marketo Engage内の IMS 組織のマッピングが必要になる場合があります。 [組織のマッピングを設定する方法については、こちらをAdobeしてください。](/help/marketo/product-docs/adobe-experience-cloud-integrations/set-up-adobe-organization-mapping.md){target=&quot;_blank&quot;}。
