# 電子メールのHTMLのダウンロード{#download-an-emails-html}

バックアップなどの目的で、Marketoでは電子メールのHTMLコンテンツをダウンロードできます。

1. 電子メールを探して選択します。 「**電子メールアクション**」ドロップダウンで、「**HTMLをダウンロード**」をクリックします。

   ![](assets/one-4.png)

簡単だ！
