# メールの HTML をダウンロードする {#download-an-emails-html}

Marketo では、バックアップやその他の目的で、メールの HTML コンテンツをダウンロードできます。

1. メールを見つけて選択します。**[!UICONTROL メールアクション]**&#x200B;ドロップダウンで、「**[!UICONTROL HTML をダウンロード]**」をクリックします。

   ![](assets/one-4.png)

簡単に設定できますね。
