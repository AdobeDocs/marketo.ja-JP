# メールの HTML をダウンロードする {#download-an-emails-html}

Marketo では、バックアップやその他の目的で、メールの HTML コンテンツをダウンロードできます。

1. メールを見つけて選択します。**メールアクション**&#x200B;ドロップダウンで、「**HTML をダウンロード**」をクリックします。

   ![](assets/one-4.png)

簡単に設定できますね。
