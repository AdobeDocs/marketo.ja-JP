# [!DNL Workplace By Facebook] を [!DNL LaunchPoint] サービスとして追加 {#add-workplace-by-facebook-as-a-launchpoint-service}

[!DNL Workplace] 統合には、次の 2 つの通知タイプが含まれます。

* **システム通知**：現在のキャンペーンのステータスに関するアラートや、即座に対処する必要のある問題（CRM エラーおよび API 制限）など、Marketo インスタンスの重要なイベントに関する [!DNL Workplace] 通知を取得します。
* **注目のアクション**：[!DNL Marketo Insight] がセールスアカウントの既知の個人によってトリガーされた場合、リードの所有者は [!DNL Workplace] 経由で通知を受け取ることができます。通知には、リード情報とセールスアカウントに関する詳細が含まれます。

>[!NOTE]
>
>**管理者権限が必要**

>[!PREREQUISITES]
>
>[!DNL Workplace] 通知が有効になっていない場合は、[Marketo サポート](https://nation.marketo.com/t5/Support/ct-p/Support){target="_blank"}にお問い合わせください。

1. 「**[!UICONTROL 管理者]**」領域に移動します。

   ![](assets/add-workplace-by-facebook-as-a-launchpoint-service-1.png)

1. **[!UICONTROL LaunchPoint]**.をクリックします。

   ![](assets/add-workplace-by-facebook-as-a-launchpoint-service-2.png)

1. 「**[!UICONTROL 新規]**」を選択し、次に「**[!UICONTROL 新規サービス]**」を選択します。

   ![](assets/add-workplace-by-facebook-as-a-launchpoint-service-3.png)

1. [!DNL Workplace] 統合の&#x200B;**[!UICONTROL 表示名]**&#x200B;を入力します。**[!UICONTROL サービス]**&#x200B;ドロップダウンで、**[!UICONTROL Workplace by Facebook]** を選択します。「**[!UICONTROL 作成]**」をクリックします。

   ![](assets/add-workplace-by-facebook-as-a-launchpoint-service-4.png)

1. システム通知や注目のアクションを受け取るには、オプションをそのままにしておきます。「**[!UICONTROL 作成]**」をクリックします。

   ![](assets/add-workplace-by-facebook-as-a-launchpoint-service-5.png)

1. 「**[!UICONTROL 承認]**」をクリックします。これにより、新しいタブで [!DNL Workplace] が開くので、[!DNL Workplace] から情報を取り込むための認証を完了し、Marketo 権限を付与します。

   ![](assets/add-workplace-by-facebook-as-a-launchpoint-service-6.png)

1. 新しい「[!DNL Workplace]」タブで、ビジネスメールまたは [!DNL Workplace] のユーザー名を入力し、「**[!UICONTROL 続行]**」をクリックします。

   ![](assets/add-workplace-by-facebook-as-a-launchpoint-service-7.png)

1. [!DNL Workplace] 資格情報を入力し、「**[!UICONTROL ログイン]**」をクリックします。

   ![](assets/add-workplace-by-facebook-as-a-launchpoint-service-8.png)

1. [!DNL Workplace] ポップアップで、Marketo からの通知を投稿する [!DNL Facebook] グループを選択します（例：[!UICONTROL パートナー統合]）。「**[!UICONTROL インストール]**」をクリックします。

   ![](assets/add-workplace-by-facebook-as-a-launchpoint-service-9.png)

1. 以下の確認通知が表示されます。タブが自動的に閉じます。

   ![](assets/add-workplace-by-facebook-as-a-launchpoint-service-10.png)

1. 「Marketo」タブを更新し、[!DNL Workplace] が [!DNL LaunchPoint] でアクティブなサービスとして表示されていることを確認します。

   ![](assets/add-workplace-by-facebook-as-a-launchpoint-service-11.png)

   手順 7 で選択した [!DNL Facebook] グループへの通知の投稿が開始されます。以下のように表示されます。

   ![](assets/add-workplace-by-facebook-as-a-launchpoint-service-12.png)
