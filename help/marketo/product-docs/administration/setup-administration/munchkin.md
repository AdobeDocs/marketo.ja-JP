# [!DNL Munchkin] {#munchkin}

[!DNL Munchkin] は、web サイトに配置する JavaScript スニペットです。訪問回数、クリック数、フォームの入力回数をトラッキングします。また、様々なクールなことを行うための豊富な API を備えています。

詳しくは、[ [!DNL Munchkin] の開発者ページ](https://experienceleague.adobe.com/en/docs/marketo-developer/marketo/javascriptapi/leadtracking/lead-tracking){target="_blank"}を参照してください。
