# Munchkin {#munchkin}

Munchkin は、Web サイトに配置する JavaScript スニペットです。訪問数、クリック数およびフォーム入力数を追跡します。 また、様々なクールな機能を実行する豊富な API を備えています。

以下を確認します。 [Munchkin の開発者ページ](https://developers.marketo.com/documentation/websites/lead-tracking-munchkin-js/){target=&quot;_blank&quot;}。
