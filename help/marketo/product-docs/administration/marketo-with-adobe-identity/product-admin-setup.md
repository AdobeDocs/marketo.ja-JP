# 製品管理者の設定 {#product-admin-setup}

1. システム管理者に招待されると、お知らせメールが届きます。 このメールで、 **はじめに**.

   ![](assets/admin-setup-7.png)

1. 以前にAdobe IDを使用してアプリケーションにアクセスしたことがある場合は、Adobe Admin Consoleにすぐに移動します。 そうでない場合、 [Adobe IDの設定](https://helpx.adobe.com/manage-account/using/create-update-adobe-id.html).

   ![](assets/admin-setup-8.png)

簡単だ！ 製品管理者は、主にユーザーの追加を担当します。 [その方法については、こちらをご覧ください](/help/marketo/product-docs/administration/marketo-with-adobe-identity/add-or-remove-a-user.md#add-a-user).
