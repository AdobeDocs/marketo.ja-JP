# ユーザーワークスペースの編集 {#edit-user-workspaces}

1. 「**[!UICONTROL 管理者]**」領域に移動します。

   ![](assets/edit-user-workspaces-1.png)

1. 「**[!UICONTROL ユーザー＆ロール]**」をクリックします。

   ![](assets/edit-user-workspaces-2.png)

1. 目的のユーザーを選択し、**[!UICONTROL ユーザーの編集]**&#x200B;をクリックします。

   ![](assets/edit-user-workspaces-3.png)

1. 必要な変更を行い、**[!UICONTROL 保存]**&#x200B;をクリックします。

   ![](assets/edit-user-workspaces-4.png)
