# [!UICONTROL 重点顧客の作成] {#create-a-named-account}

重点顧客を手動で作成するには、次の手順に従います。

1. 「[!UICONTROL 重点顧客]」で、「**[!UICONTROL 新規]**」ドロップダウンをクリックし、「**[!UICONTROL 重点顧客を作成]**」を選択します。

   ![](assets/two-1.png)

1. 目的のフィールドに入力し、「**[!UICONTROL 作成]**」をクリックします。

   ![](assets/three-1.png)

   >[!TIP]
   >
   >重点顧客ダッシュボードを表示するにはその重点顧客を直接クリックします。
