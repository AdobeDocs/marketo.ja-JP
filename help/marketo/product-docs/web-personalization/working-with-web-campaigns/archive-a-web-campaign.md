# Webキャンペーンのアーカイブ {#archive-a-web-campaign}

1. 「 **Webキャンペーン**」に移動。

   ![](assets/one.jpg)

1. 目的のWebキャンペーンーのステータスドロップダウンをクリックし、「 **アーカイブ**」を選択します。

   ![](assets/two-3.png)

   >[!NOTE]
   >
   >アーカイブ済みWebキャンペーンは、デフォルトのフィルターに表示されません。 ユーザーを表示するには、フィルターアイコンをクリックし、「 **ステータス**」で「 **アーカイブ済み** 」チェックボックスを選択し、「 **適用**」をクリックします。
