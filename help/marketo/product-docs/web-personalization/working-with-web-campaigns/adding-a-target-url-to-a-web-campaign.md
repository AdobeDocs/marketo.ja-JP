# Web キャンペーンへのターゲット URL の追加 {#adding-a-target-url-to-a-web-campaign}

Target URL はキャンペーンを設定ページの下に配置され、Web キャンペーンを表示する特定の URL または URL を定義します。

## ダイアログまたはウィジェット Web キャンペーン用のターゲット URL の追加 {#adding-a-target-url-for-dialog-or-widget-web-campaigns}

1. 「**Web キャンペーン**」に移動します

   ![](assets/web-campaigns-hand-5.jpg)

1. 「**Web キャンペーンの新規作成**」を選択します。

   ![](assets/create-new-web-campaign-hand.jpg)

1. を追加します。 **キャンペーン名**. を選択します。 **ターゲットセグメント**. 追加 **ターゲット URL**.

   ![](assets/set-web-campaign-hands.jpg)

## Target URL への複数の URL の追加 {#adding-multiple-urls-to-target-url}

プラスアイコン (![—](assets/image2015-2-18-8-3a40-3a59.png)) をクリックすると、複数の値を入力ダイアログが開き、複数の URL を追加できます。 1 行に 1 つの URL を追加します。

![](assets/image2015-2-23-18-3a15-3a57.png)

>[!NOTE]
>
>* ダイアログおよびウィジェットの Web キャンペーンでは、「任意のページ」および「ワイルドカード (*) 」オプションを使用できます。
* 高度な使用例では、ゾーン内 Web キャンペーンで URL パスの末尾にワイルドカードを使用できます。 例： [www.marketo.com/software/personalization/*](https://www.marketo.com/software/web-personalization/)
* URL では大文字と小文字が区別されます


## ゾーン内ウェブキャンペーン用のターゲット URL の追加 {#adding-a-target-url-for-in-zone-web-campaigns}

1. に移動します。 **Web** **キャンペーン**.

   ![](assets/web-campaigns-hand-5.jpg)

1. 「**Web キャンペーンの新規作成**」を選択します。

   ![](assets/create-new-web-campaign-hand.jpg)

1. を追加します。 **キャンペーン名**. を選択します。 **ターゲットセグメント**. 追加 **ターゲット URL**.

   >[!NOTE]
   ゾーン内の Target URL は、特定の URL または URL を定義する必要があります。 高度な使用例では、ゾーン内 Web キャンペーンで URL パスの末尾にワイルドカードを使用できます。 例： [www.marketo.com/software/personalization/*](https://www.marketo.com/software/web-personalization/)

   ![](assets/set-web-campaign-multiple-hands.jpg)
