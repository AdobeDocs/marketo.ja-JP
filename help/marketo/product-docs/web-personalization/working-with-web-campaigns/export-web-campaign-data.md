# Web キャンペーンデータをエクスポートする {#export-web-campaign-data}

次の簡単な手順に従って、Web キャンペーンデータをエクスポートします。

1. 「**[!UICONTROL Web キャンペーン]**」に移動します

   ![](assets/one-2.png)

1. ページの右上にある  CSV を書き出しアイコンをクリックします。

   ![](assets/two-2.png)

1. ファイルを開くか、保存します。

   ![](assets/three-2.png)

1. 役に立つ統計を確認するには、ファイルを表示します。

   ![](assets/four-1.png)
