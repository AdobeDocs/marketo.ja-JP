# Webキャンペーンを最新または最高パフォーマンス{#sort-web-campaigns-by-latest-or-top-performing}で並べ替え

Webキャンペーンーは、いくつかの異なる方法で並べ替えることができます。

1. **Webキャンペーン**&#x200B;に移動します。

   ![](assets/web-campaigns-hand-1.jpg)

1. 並べ替えタイプを選択します。

   >[!NOTE]
   >
   >**定義**
   >
   >**最新** -キャンペーンが作成された日付に基づいて並べ替えが行われます。最新のキャンペーンが先頭に表示されます。
   >
   >**パフォーマンスが最も高い**  — クリックスルー率に基づいてキャンペーンを並べ替えます。上部のクリックスルー数が最も多い。

   ![](assets/image2016-11-4-13-3a34-3a59.png)

   はい、本当に簡単です。
