# 地域の編集 {#edit-regions}

ユーザーの地域設定を変更して、特定の地域のデータのみを表示したい場合

1. **ユーザー設定**&#x200B;に移動します。

   ![](assets/image2014-12-1-23-3a8-3a40.png)

1. [**領域の編集**]をクリックします。

   ![](assets/image2014-12-3-18-3a55-3a25.png)

1. 地域に関連する国または州を確認します。

>[!NOTE]
>
>米国を選択すると、ページの下部に開き、米国の州のオプションがすべて選択されます。
