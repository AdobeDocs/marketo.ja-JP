# 地域を編集する {#edit-regions}

特定の地域のデータのみを表示するようにユーザーの地域設定を変更したい場合は、

1. **ユーザー設定**&#x200B;に移動します。

   ![](assets/image2014-12-1-23-3a8-3a40.png)

1. 「**地域の編集**」をクリックします。

   ![](assets/image2014-12-3-18-3a55-3a25.png)

1. お住まいの地域に関連する国や州を確認してください。

>[!NOTE]
>
>米国を選択すると、ページの下部に、選択するすべての米国の州のオプションが開きます。
