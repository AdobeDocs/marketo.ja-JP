# Web パーソナライゼーション用語集 {#web-personalization-glossary}

Marketo Web パーソナライゼーションの世界と言語に関するインサイトがいくつかあります。

| 用語 | 定義 |
|---|---|
| **匿名訪問者** | フォームに入力したこともウェブサイトに詳細を残したこともないウェブ訪問者。 |
| **ウェブキャンペーン** | 特定のセグメントに関連付けられたカスタマイズされたリアクション。Web パーソナライゼーションでは、Web キャンペーンには、ダイアログ、ゾーンおよびウィジェットが含まれます。 |
| **クリックストリーム** | 訪問者のサイト上のアクティビティと URL パス、および各ページの訪問期間 |
| **ISP** | インターネットサービスプロバイダ |
| **既知の訪問者** | フォームに入力し、Web サイトに詳細（電子メールアドレス）を残した、またはMarketoの電子メールのリンクをクリックした Web 訪問者。 |
| **顧客リスト** | 主要なアカウント名／組織名のリスト。アカウントベースドマーケティング（ABM）リストとも呼ばれます。 |
| **セグメント** | 指定した条件に一致する訪問者のコレクション ( [「セグメントを設定」ページ](/help/marketo/product-docs/web-personalization/using-web-segments/web-segments.md). |
| **分割テスト** | 2 つ以上のバリエーションを用いたテスト実験で、結果の違いを測定します。目標は、関心のある結果を増加または最大化させるウェブページへの変更を特定することです。 |
| **ワイルドカード** | ワイルドカード文字 (&#42; を使用する ) の前後に使用され、文字列内の他の文字や文字に置き換えられます。 以下の例を参照してください。 |

## ワイルドカードの例 {#wildcard-examples}

Web パーソナライゼーションでワイルドカードを使用する方法は 3 つあります。

価格で終わるページ URL のすべての訪問者を一致させます ( 例： `www.marketo.com/pricing`

![](assets/wildcard-example-1.png)

https://で始まるページ URL のすべての訪問者を一致させます ( 例： `https://www.marketo.com`

![](assets/wildcard-example-2.png)

backup という単語を含むページ URL のすべての訪問者を一致させます ( 例： `https://www.marketo.com/backup/pricing.html`

![](assets/wildcard-example-3.png)
