# Marketo AIの概要 {#overview}

Marketoの会話型AIは、面倒だが重要なマーケティング機能を自動化するために設計された、多くのエージェントを提供します。

## アクセス方法 {#access}

マイMarketo画面で、**AIでビルド** タイルをクリックします。

![](assets/overview-1.png)

会話型AI UI インターフェイスは、生産性を向上させるために手作業を自動化するように設計された、一連のエージェント機能を提供します。

ダイアログプロンプト領域の「**基本を学ぶ**」ボタンをクリックします（「_AI エージェントを使い始める…_」はデフォルトでプロンプトに表示されます）。

![](assets/overview-2.png)

センターコンソールには、さまざまなタスクに役立つ7つのエージェント機能があります。

## エージェント {#agents}

### リードを調査（近日リリース予定） {#investigate-lead}

MQL、プログラムの選定基準、ライフサイクルの進捗を確認しなかった理由を説明します。

### プランプログラム（近日リリース予定） {#plan-program}

キャンペーン概要から、他のユーザーが使用するプログラム設定ドキュメントを作成します。

### プログラムの作成（近日リリース予定） {#create-program}

電子メール、ランディングページ、スマートキャンペーンを含むキャンペーン概要を使用して、Marketo Engageプログラム全体を生成します。

### プログラム QA {#program-qa}

Marketo Engageプログラムをベストプラクティスと比較して検証し、問題がある場合はローンチ前に発見します。 [詳細情報](/help/marketo/product-docs/marketo-ai/agents/program-qa.md)

### リードの読み込み {#import-leads}

フィールドマッピング機能を利用すれば、リードリストをMarketo Engageデータベースにインポートして重複を排除できます。 [詳細情報](/help/marketo/product-docs/marketo-ai/agents/import-leads.md)

### データの標準化（近日リリース予定） {#normalize-data}

企業名、役職、国などのフィールドを標準化します。

### 呼び出し可能なエージェント（近日リリース予定） {#callable-agents}

これらのエージェントは、リアルタイムのデータ処理のために、Marketo Engage Smart Campaignsでwebhookとして実行されます。
