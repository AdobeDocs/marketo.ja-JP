# リードの読み込み {#import-leads}

フィールドマッピング機能を利用すれば、リードリストをMarketo Engageデータベースにインポートして重複を排除できます。

リードのリストを読み込みますか？ 人々？ csv形式で、データの正規化ルールを適用してMarketo Databaseに保存します。

このエージェントをクリックすると、対話型AIに移動します。 左側のパネルには、エージェントが実行するデータ正規化機能に関するガイダンス、応答、オプションを投稿するためのウィンドウを提供するAI アシスタントがあります。

インポートプロセスを開始するには、添付ファイルのアイコンをクリックしてファイルをアップロードします。

「インポートリスト」に入力し、「送信」をクリックします

適用するビジネスルールを入力し、「送信」をクリックします。 入力すると、センターコンソールのウィンドウに、適用されたルールの結果が表示されます。

必要に応じて別のビジネスルールを入力し、「送信」をクリックします。

マッピングされたリストフィールドのリストをデータベースに表示するには、「マッピング」タブをクリックします。

データベースに保存する準備ができたら、「Marketoに読み込む」をクリックします。

リストを保存するフォルダーを選択し、リードリストの名前を入力して同意ボックスにチェックを入れ、「承認してMarketoに読み込む」ボタンをクリックします。

リストを保存すると、処理済み、失敗した、または警告が発生したレコードの概要が検証に提供されます。
