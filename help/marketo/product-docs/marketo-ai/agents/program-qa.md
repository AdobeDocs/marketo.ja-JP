# プログラム QA {#program-qa}

メール、ランディングページ、キャンペーンなど、あらゆるコンポーネントをまたいでベストプラクティスのプログラムを監査します。

このエージェントにアクセスするには2つの方法があります。

中央のウィンドウからエージェントをクリックします右上のドロップダウンをクリックします

「Use this Agent」ボタンをクリックします

プロンプトダイアログボックスの左側のペインに、デフォルトのテキストが表示されます。

次に、右側のペインでQAするプログラムを選択します

プログラムの概要が中央のウィンドウに表示され、プログラムの概要とその作成内容が表示されます。

または、会話型AIのメイン画面で「エージェント」をクリックします。

AI アシスタントには、実行されるエージェントを示すデフォルトテキストが表示されます。

次に、右側のQAが必要なプログラムを選択します。

同じ概要がセンターコンソールに表示されます
