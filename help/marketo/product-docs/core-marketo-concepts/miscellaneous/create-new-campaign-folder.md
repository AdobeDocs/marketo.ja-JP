# 新しいキャンペーンフォルダーの作成 {#create-new-campaign-folder}

キャンペーンフォルダーを使用すると、Tidyワークスペースを整理しやすくなります。 次の簡単な手順に従うと、自動化の魔法を使うことができます。

1. 「 **マーケティングアクティビティ**」に移動します。

   ![](assets/login-marketing-activities.png)

1. 「 **新規**」を選択します。

   ![](assets/image2015-2-25-7-3a57-3a18.png)

1. 「 **新規キャンペーンフォルダー**」を選択します。

   ![](assets/image2015-2-25-7-3a58-3a15.png)

1. キャンペーンフォルダの **名前** (Name)を入力します。

   ![](assets/image2015-2-25-8-3a0-3a20.png)

1. オプション：「 **説明** 」を入力し、「**作成」をクリックします。**

   >[!TIP]
   >
   >説明は、購読上の他のユーザー向けです。 このメッセージはユーザーに表示されません。

   ![](assets/image2015-2-25-8-3a9-3a3.png)

   すごい！ キャンペーンフォルダーを作成したとき。 ツリーで調べてみてください。

   ![](assets/image2015-2-25-8-3a10-3a29.png)

   新しいプログラムを [作成すると](../../../product-docs/core-marketo-concepts/programs/creating-programs/create-a-program.md)、このキャンペーンフォルダがオプションとして表示されます。
