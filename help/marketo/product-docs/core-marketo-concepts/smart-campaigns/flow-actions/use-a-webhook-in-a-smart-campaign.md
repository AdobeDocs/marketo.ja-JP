# スマートキャンペーンでの web フックの使用 {#use-a-webhook-in-a-smart-campaign}

[Web フック](https://developers.marketo.com/documentation/webhooks/)を使用するには、フローアクションとして[スマートキャンペーン](/help/marketo/product-docs/core-marketo-concepts/smart-campaigns/flow-actions/add-a-flow-step-to-a-smart-campaign.md)に追加します。

>[!AVAILABILITY]
>
>すべてのお客様がこの機能を購入しているわけではありません。詳細は、セールス担当者にお問い合わせください。

1. [スマートキャンペーンの作成](/help/marketo/product-docs/core-marketo-concepts/smart-campaigns/creating-a-smart-campaign/create-a-new-smart-campaign.md)。

   >[!NOTE]
   >
   >Web フックは、キャンペーンでのみトリガーできます。

1. 「**フロー**」タブに移動し、「**Web フックを呼び出し**」フローアクションをドラッグします。

   ![](assets/image2014-9-22-15-3a8-3a2.png)

1. Web フックを選択します。

   ![](assets/image2014-9-22-15-3a8-3a5.png)

1. スマートリストでも web フックを使用できます。

   ![](assets/2017-05-02-10-54-38.png)

1. 最後に、フローステップの「**選択肢を追加**」で使用できます。

   ![](assets/image2014-9-22-15-3a8-3a13.png)
