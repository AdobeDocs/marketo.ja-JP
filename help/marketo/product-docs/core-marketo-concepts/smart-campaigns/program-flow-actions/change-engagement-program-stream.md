# エンゲージメントプログラムストリームの変更 {#change-engagement-program-stream}

## 概要 {#overview}

通常は、これを実現するために[ストリームトランジションルール](/help/marketo/product-docs/email-marketing/drip-nurturing/engagement-program-streams/transition-people-between-engagement-streams.md)を使用しますが、あるストリームから別のストリームに人物を手動で移動する場合は、このフローステップを使用します。

![](assets/image2014-9-22-14-3a52-3a14.png)

## 使用方法 {#usage}

1. まず、人物を移動するエンゲージメントプログラムを選択します。

   >[!NOTE]
   >
   >別のプログラムを選択すると、その人物は現在のストリームに残り、新しいプログラムに追加されます。

   ![](assets/image2014-9-22-14-3a52-3a50.png)

1. 人物を追加するストリームを選択します。

   ![](assets/image2014-9-22-14-3a52-3a59.png)

これで完了です。
