# マーケティングカレンダーライセンスの発行と取り消し {#issue-revoke-a-marketing-calendar-license}

>[!NOTE]
>
>**必要な管理者権限**

マー [ケティングカレンダーのシートを利用するには](http://docs.marketo.com/display/docs/marketing+calendar) 、アクセス権を必要とするユーザーにライセンスを発行する必要があります。 これが方法です。

1. 「 **管理者** 」セクションに移動します。

   ![](assets/adminhand.png)

1. 「 **ユーザーと役割**」をクリックします。

   ![](assets/2.png)

1. ユーザーを選択し、「 **Issue License」をクリックします。**

   >[!TIP]
   >
   >1回の操作で複数のユーザーを選択するには、 **CtrlまたはCommandキーを押しながら** 「クリック」します。

   ![](assets/3.png)

1. 「 **Enable License** 」を選択し、「 **Save」をクリックします。**

   >[!NOTE]
   >
   >ライセンス数は5個に制限されています。 詳しくは、担当営業にお問い合わせください。

   ![](assets/4.png)

   うまくいった！ [ **カレンダー**?]の下の緑のチェックマークが表示されます。

   ![](assets/5.png)

Awesome — 予定を立てる準備ができました。
