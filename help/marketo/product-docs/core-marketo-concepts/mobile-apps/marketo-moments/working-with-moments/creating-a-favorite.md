# お気に入りの作成 {#creating-a-favorite}

将来の参照用にカードをお気に入りに登録します。

1. カードメニューを開きます。

   ![](assets/image2015-7-14-16-3a28-3a54.png)

1. 「お気に入り」をタップします。

   ![](assets/image2015-7-14-16-3a36-3a22.png)

   それだけだ！
