# Marketo Moments のパーソナライズ {#personalizing-marketo-moments}

マーケティングプログラムやスマートキャンペーンが多数ある場合、すべてを表示できるのも素晴らしいですが、自分の作業にのみ集中したい場合があります。

「**自分のモーメントのみを表示**」を有効にして、独自のメールプログラムとスマートキャンペーンのみを表示します。

![](assets/image2015-7-16-15-3a53-3a24.png)

または、「**自分のモーメントのみを表示**」を無効にして、アクセス権のあるすべてのスマートキャンペーンとメールプログラムを表示します。

![](assets/image2015-7-16-15-3a55-3a29.png)
