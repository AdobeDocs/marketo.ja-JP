# 手動での人物の作成 {#create-a-person-manually}

人をMarketoに入れるには、様々な方法があります。 手動で作成する必要がある場合は、次の方法を実行します。

1. データ **ベースに移動します**。

   ![](assets/db-1.png)

1. 「 **新規**」で、「 **新規ユーザー**」をクリックします。

   ![](assets/two-2.png)

1. ユーザーの情報を入力し、「 **作成**」をクリックします。

   ![](assets/three-2.png)

これは、Marketoに1人のユーザーを簡単に追加する方法です。
