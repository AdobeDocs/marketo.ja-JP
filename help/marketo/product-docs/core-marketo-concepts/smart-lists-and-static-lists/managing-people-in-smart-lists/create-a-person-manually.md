# 人を手動で作成{#create-a-person-manually}

人をMarketoに入れる方法はいろいろある。 手動で作成する必要がある場合は、次の方法を実行します。

1. **データベース**&#x200B;に移動します。

   ![](assets/db-1.png)

1. 「**新しい**」で、「**新しい人**」をクリックします。

   ![](assets/two-2.png)

1. ユーザーの情報を入力し、「**作成**」をクリックします。

   ![](assets/three-2.png)

Marketoに一人を加える簡単な方法だ！
