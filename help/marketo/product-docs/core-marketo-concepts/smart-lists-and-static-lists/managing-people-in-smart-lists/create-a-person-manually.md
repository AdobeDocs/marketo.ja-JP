# リードの手動作成 {#create-a-person-manually}

リードを Marketo に入れる方法はいろいろあります。手動で作成する必要がある場合は、次の手順に従います。

1. **データベース**&#x200B;に移動します。

   ![](assets/db-1.png)

1. 「**新規作成**」で「**新規リード**」をクリックします。

   ![](assets/two-2.png)

1. リードの情報を入力し、「**作成**」をクリックします。

   ![](assets/three-2.png)

Marketo に 1 件のリードを追加するには、この方法が手早く簡単です。
