# リストまたはスマートリストの更新 {#refresh-a-list-or-smart-list}

スマートリストを実行し、数分が経過した場合、結果は今は異なる場合があります — 更新して調べてください。

## 結果の更新 {#refresh-results}

1. 内のデータを更新するには、以下を実行します。 **[!UICONTROL People]** スマートリストのタブで、更新アイコンをクリックします。

   ![](assets/refreshbutton.png)

1. スマートリストが再実行され、より最新の結果セットが表示されます。

   ![](assets/loadingrefresh.png)

>[!TIP]
>
>スマートリストを実行し、後で再び表示すると、右下隅の人数の前に「概要」という単語が表示される場合があります。 件数を更新するには、件数の表示そのものをクリックします。最新の正確な件数が表示されます。
