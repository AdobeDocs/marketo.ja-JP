# リストまたはスマートリストの更新 {#refresh-a-list-or-smart-list}

スマートリストを実行して数分が経過した場合、結果は異なる可能性があります。更新して確認します。

## 結果の更新 {#refresh-results}

1. スマート・リストの **[!UICONTROL ユーザー]** タブのデータを更新するには、更新アイコンをクリックします。

   ![](assets/refreshbutton.png)

1. スマート・リストが再実行され、より最新の結果セットが表示されます。

   ![](assets/loadingrefresh.png)

>[!TIP]
>
>スマートリストを実行して後で戻ったときに、右下隅の人物カウントの前に「バージョン情報」という単語が表示される場合があります。 件数を更新するには、件数の表示そのものをクリックします。最新の正確な件数が表示されます。
