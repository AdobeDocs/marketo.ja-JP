# プログラムのローカルアセットの理解 {#understanding-local-assets-in-a-program}

ローカルアセットは、プログラムを構成するコンテンツです。アセットはカスタマイズ可能で、自動化されたマーケティング施策を構築できます。プログラムで作成できるローカルアセットのほぼすべてを次に示します。

![](assets/one.png)
