# 追加的な管理者 {#additional-admins}

チームメンバーを変更または削除するには、[!UICONTROL 設定]ページに移動し、左ナビゲーションパネルの「**[!UICONTROL チーム管理]**」をクリックします。
アカウントの管理者は、チームのメンバーを管理できます。
