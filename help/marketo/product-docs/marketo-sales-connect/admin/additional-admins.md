# 追加の管理者 {#additional-admins}

チームメンバーを変更または削除するには、設定ページに移動し、左側のナビゲーションパネルで「 **チーム管理*** *」をクリックします。
アカウントの管理者は、ここでチーム内のユーザーを管理できます。
