# アクティビティ履歴にログが記録されなかった {#didnt-log-to-activity-history}

通話が [!DNL Salesforce] に記録されない場合は、[!DNL Salesforce] でリード/連絡先が見つからないことが原因である可能性があります。

問い合わせが正しいリード/連絡先に記録され [!DNL Sales Connect] ようにするには、Salesforce ID が必要です。 Salesforce ID を [!DNL Sales Connect] に追加する最善の方法は、[Sales Connect へのプッシュ ](/help/marketo/product-docs/marketo-sales-connect/crm/salesforce-customization/push-to-sales-connect.md) 機能を使用することです。 プッシュを使用して [!DNL Sales Connect] に移行する場合、[!DNL Salesforce] ID が連絡先の移行に含まれます。
