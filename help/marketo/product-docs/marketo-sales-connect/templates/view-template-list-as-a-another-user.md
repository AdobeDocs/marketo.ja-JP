# 別の表示としてのユーザーテンプレートリスト {#view-template-list-as-a-another-user}

管理者は、任意のユーザーとして表示テンプレートを使用できます。

>[!NOTE]
>
>**必要な管理者権限**

1. 「 **テンプレート**」をクリックします。

   ![](assets/one.png)

1. 「 **表示名** 」ドロップダウンをクリックし、目的のユーザーを選択します。

   ![](assets/two.png)

1. これで、選択したユーザーとしてテンプレートが表示されます。

   ![](assets/three.png)

   >[!NOTE]
   >
   >また、フィルターや検索機能を表示Asと共に使用して、最も関連のあるものを表示することもできます。
