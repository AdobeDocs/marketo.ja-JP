# 動的フィールドで自分自身を参照{#reference-yourself-in-a-dynamic-field}

動的フィールドは差し込み印刷の形式です。

Eメールテンプレートを、`{{my_name}}`や`{{my_title}}`などの定義済みの属性でパーソナライズできます。 これらのフィールドを使用すると、電子メールテンプレート内で自分をすばやく参照できます。

テンプレートに自己参照用の動的フィールドを追加するには、テンプレートエディターの「**MSE動的フィールド**」ボタンから、使用したいフィールドを選択します。
