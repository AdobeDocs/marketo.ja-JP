# 動的フィールドでの自分の参照 {#reference-yourself-in-a-dynamic-field}

動的フィールドは差し込み印刷の形式です。

またはのような事前定義済みの属性を使用して、電子メールテンプレートをパーソナライズ `{{my_name}}` でき `{{my_title}}`ます。 これらのフィールドを使用すると、電子メールテンプレート内で自分をすばやく参照できます。

自己参照用の動的フィールドをテンプレートに追加するには、テンプレートエディターの **MSE動的フィールド** (MSE Dynamic Fields)ボタンから、使用したいフィールドを選択します。
