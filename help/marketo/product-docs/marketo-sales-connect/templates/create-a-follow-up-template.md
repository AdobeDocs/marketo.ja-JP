# フォローアップテンプレートの作成 {#create-a-follow-up-template}

キャンペーン機能を使用して、新しいテンプレートを作成する場合は、「last_email_body」動的フィールドをテンプレートの下部に追加します。 それでキャンペーン内の電子メールが自動的に繋がります。
