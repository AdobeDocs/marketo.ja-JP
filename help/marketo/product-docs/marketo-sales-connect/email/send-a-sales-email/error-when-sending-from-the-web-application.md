# Web アプリケーションから送信する際のエラー {#error-when-sending-from-the-web-application}

通常、 [Webアプリケーションからエラーが発生した場合は](http://toutapp.com/login)、電子メールの形式設定または動的フィールドの問題が原因です。 構成ウィンドウですべてが正しく入力されていることを確認し、もう一度送信を試みてください。
