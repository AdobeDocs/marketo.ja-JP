# Web アプリケーション{#error-when-sending-from-the-web-application}から送信する際のエラー

通常、[Webアプリケーション](https://toutapp.com/login)からエラーが発生した場合は、電子メールまたは動的フィールドの形式設定の問題が原因です。 構成ウィンドウですべてが正しく入力されていることを確認し、もう一度送信を試みてください。
