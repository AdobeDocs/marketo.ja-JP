# リリースノート：2015年6月{#release-notes-june}

2015年6月のリリースには、次の機能が含まれています。 Marketing Editionで機能が利用できるかどうかを確認してください。 リリース後は、各機能の詳細記事へのリンクを必ず戻して参照してください。

## [アトリビューション電子メールレポート](../../product-docs/web-personalization/reporting-for-web-personalization/email-reports.md)  {#attribution-email-report}

マーケティングアクティビティに提供される価値のパーソナライゼーションとレコメンデーションコンテンツを確認します。 [アトリビューション電子メール](../../product-docs/web-personalization/reporting-for-web-personalization/email-reports.md) レポートには、RTPのパーソナライゼーションと推奨コンテンツキャンペーンから生じた直接リードおよび補助リードが表示されます。「RTP’s, User Settings and Email Report」で、月次または四半期別の電子メールを受信するアトリビューション電子メールレポートを追加します。

## ロードマップに影響を与えたいか？{#want-to-influence-our-roadmap}

[クイックレポートカード](https://www.surveymonkey.com/s/VG9YCT5)を使用して、グレードマーケティングを行います。 2016年以降の計画を立てるために、お客様のご意見をお寄せください。
