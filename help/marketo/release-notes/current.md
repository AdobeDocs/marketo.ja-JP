# リリースノート：2024年3月 {#release-notes-mar-24}

以下に、24 年 3 月リリースに含まれるすべての機能を示します。 機能の可用性についてはお使いの Marketo Engage のエディションをご確認ください。

>[!AVAILABILITY]
