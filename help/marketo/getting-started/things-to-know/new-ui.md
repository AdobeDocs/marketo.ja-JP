# Marketo Engageの新しいUI {#new-ui}

新しいMarketo Engage UI ベータ版にご参加いただきありがとうございます。 このアップデートは、Marketo Engageのスタイル設定を最新化し、機能を変更することなくレスポンシブを改善します。 新しいUIには、Marketo Engageのほとんどのページの右上隅に表示されるドロップダウンを使用してアクセスします。

## 事前準備 {#before-starting}

新しいUIにアクセスする前に、以下が必要です。

* 1つ以上のMarketo Engage ユーザーロールに対する&#x200B;_新しいUIへのアクセス_&#x200B;権限が付与されました。

* 求められたら、オープンベータテストの条件に同意します。

## 新しいUI権限 {#new-ui-permission}

管理者は、1つ以上のユーザー役割に&#x200B;_新しいUIへのアクセス_&#x200B;権限を付与できます。

1. **管理者**&#x200B;領域で、**ユーザーと役割**&#x200B;を選択します。

   ![](assets/new-ui-1.png)

1. 「**ロール**」タブをクリックします。 目的の役割を選択し、**役割を編集**&#x200B;をクリックします。

   ![](assets/new-ui-2.png)

>[!NOTE]
>
>新しい役割を作成することもできます。

1. 「**新しいテーマにアクセス**」チェックボックスを選択し、**保存**&#x200B;をクリックします。

   ![](assets/new-ui-3.png)

## 新規UIとクラシック UI {#new-and-classic}

新しいUIに切り替えるには、右上隅のUI ドロップダウンをクリックし、**新規（Beta）**&#x200B;を選択します。

![](assets/new-ui-4.png)

何らかの理由で元に戻す必要がある場合は、UI ドロップダウンをもう一度クリックし、**Classic**&#x200B;を選択します。

## フィードバックの送信 {#feedback}

ぜひフィードバックをお寄せください。 新しいUIの調査中に機能へのアクセスや使用で問題が発生した場合、または提案や懸念がある場合は、右上の「**UI Beta フィードバック**」ボタンをクリックします。

![](assets/new-ui-5.png)
