# 新しい領域：マーケティングアクティビティのチェックリスト {#new-area-marketing-activities-checklist}

マーケティングアクティビティには、自動化されたマーケティングプログラムを構成するすべてのアセットとコンテンツが含まれます。新しいMarketo Engageインスタンスを設定する際に、クリーンで明確な組織により、すべてのユーザーが様々なプログラムを簡単に見つけて管理できるようになります。 設定の進行状況を追跡し、ダウンロード可能なチェックリストを使用してオフラインで重要な手順に従います - リンクはこちら -
