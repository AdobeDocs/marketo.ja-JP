# 助けてください！既存の Marketo インスタンスを引き継ぎました

**既存の Marketo インスタンスを引き継いだばかりで、どこから始めればよいのか途方に暮れていませんか？**

もしそうであれば、このガイドは、あなたのためのものです。必要なものを得るために正しい方向を示すことで、Marketo を使いはじめて最初の数日間を最大限に活用するのに役立ちます。
