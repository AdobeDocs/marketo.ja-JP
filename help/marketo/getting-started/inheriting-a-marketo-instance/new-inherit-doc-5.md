# ドキュメント 5 を継承 {#inherit-doc-5}

これで、確立されたインスタンスを継承する際に監査する主な製品領域を学びました。次の手順は、インスタンス設定とテクニカルスタックに関するドキュメントを作成/更新することです。 スプレッドシートやプロジェクト管理アプリケーションを使用して作成する場合でも、ドキュメントを参照すれば、進行状況を追跡し、詳細を記録するのに役立ちます。
